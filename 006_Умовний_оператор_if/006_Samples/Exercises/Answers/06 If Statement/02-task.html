Завдання:
        Допрацюйте сценарій.
        Якщо користувач ввів значення 0 або не числове значення для висоти та радіуса циліндра - повідомте про помилку.
        Якщо введені значення коректні – розрахуйте об'єм циліндра.
        Якщо конвертація рядкового значення в ціле чисельне пройшла не успішно, то результат конвертації NaN. Для перевірок
        значення за умови рівності з NaN потрібно використовувати функцію isNaN(value), яка поверне true, якщо значення value не
        є числом.
        Така перевірка – value == NaN, не спрацює.
